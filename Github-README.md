Repositório de Prática – Curso de Git e GitHub

Este repositório foi utilizado como ambiente de prática durante a realização de um curso sobre Git e GitHub. O objetivo principal foi aplicar, de forma prática, os 
conceitos apresentados no curso, realizando commits reais em um projeto de código, em vez de executar apenas commits vazios ou aleatórios.

Para isso, foi utilizado um projeto base já existente, que serviu como estrutura para simular um fluxo real de desenvolvimento e versionamento de código.

Comandos Básicos do Git

Abaixo estão alguns dos principais comandos utilizados durante o processo de aprendizado:

Inicializar um repositório Git
git init
Verificar o estado dos arquivos
git status
Adicionar arquivos para commit
git add .

ou

git add nome-do-arquivo
Criar um commit
git commit -m "mensagem do commit"
Ver histórico de commits
git log
Conectar o repositório local a um repositório remoto
git remote add origin URL_DO_REPOSITORIO
Enviar commits para o GitHub
git push origin main
Atualizar o repositório local com alterações remotas
git pull origin main
